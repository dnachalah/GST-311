Question 1
Write 20 HTML tags and what they are used for

1. The "(p)(/p)" tag.
This creates a paragraph in the html document.
2. The "(b)(/b)" tag.
This boldens a selected text in the html document.
3. The "(i)(/i)" tag.
This italicizes a selected text in the html document.
4. The "(body)(/body)" tag.
This defines the document's body.
5. The "(br)" tag.
This creates a single line break.
6. The "(a)(/a)" tag.
This creates a hyperlink in the html document.
7. The "(div)(/div)" tag.
This defines a section in the html document.
8. The "(dl)(/dl)" tag.
This creates a description list in the html document.
9. The "(form)(/form)" tag.
This creates a form for user input.
10. The "(h1 - h6)(/h1 - h6)" tag.
These define HTML headings.
11. The "(header)(/header)" tag.
This defines a header for a document or section.
12. The "(html)(/html)" tag.
This defines the root of an HTML document.
13. The "(img)" tag.
This defines an image in the html document.
14. The "(ins)(/ins)" tag.
This defines a word or text has been inserted in the html document.
15. The "(li)(/li)" tag.
This defines a list item.
16. The "(link)(/link)" tag.
This defines the relationship between a document and an external resource.
17. The "(q)(/q)" tag.
This defines a short quotation in the html document.
18. The "(strong)(/strong)" tag.
This defines a important text in the html document.
19. The "(table)(/table)" tag.
This creates a table in the html document.
20. The "(sup)(/sup)" tag.
This defines a superscripted text in the html document.

Question 2
Write the different types of list we have and explain briefly their use cases.

We have 3 commonly used lists, namely:

1. Ordered Lists
This list is enclosed in the (ol)(/ol) tag. Each list item is enclosed in the (li)(/li) tag.
The list items will be marked with numbers by default, but can be changed to roman numerals or alphabets with certain attributes
2. Unordered Lists
This list is enclosed in the (ul)(/ul) tag. Each list item is enclosed in the (li)(/li) tag.
The list items will be marked with dots by default, but can be changed to different shapes with certain attributes
3. Description Lists
This list is a list of terms, with a description of each term.
This list is enclosed in the (dl)(/dl) tag. Each term(name) is enclosed in the (dt)(/dt) tag.
The description of each term is enclosed in the (dd)(/dd) tag.

How does host, domain work.

A domain host provides a domain name, like www.yourdomain.com, that visitors can use to find you. A domain name is like a street address that directs people to your website's location, but it's not the content that visitors see when they visit your site.
